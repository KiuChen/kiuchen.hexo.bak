一、入门

  1. JDK和JRE

    • JDK：JRE + Java核心开发工具(javac.exe、java.exe、javadoc.exe)
    • JRE：JVM + Java核心类库

  2. 为什么配置Path环境变量呢？

    • 刚安装好的JAVA只能在其bin目录下执行java命令，为了让电脑上任何地方都可以执行Java命令，就得配置Path环境变量

    • JAVA_HOME = bin的上一层目录

      path = JAVA_HOME%\bin

  3. 编译和运行.java文件

    • 编译：javac abc.java 编译错误，一般是语法问题
    • 运行：java abc

  4. char c = ‘’; //空字符，编译不通过

    String s = “”; //空字符串，能够正常运行

  5. 标识符命名规范：

    • 包名：aabbcc 全部小写
    • 类名、接口名：AaBbCc 单词首字母大写
    • 变量名、方法名：aaBbCc 除首字母外其他首字母大写 即驼峰命名法
    • 常量名：AA_BB_CC 全大写，单词间用下划线_隔开

  6. 八种基本类型变量、引用数据类型：

    • boolean、byte(1)、char(2) 、short(2)、int(4)、long(8)、float(4)、double(8)
    • 类、接口、数组

  7. 数据类型之间的自动转换(小到大)

    • byte、shrot、char->int->long->float->double

  8. 获取系统输入：

    • Scanner sc = new Scanner(System.in)
      • sc.next()获取下一个字符串
      • sc.nextInt()获取下一个int数据
      • sc.nextDouble()获取下一个double数据
      • sc.nextBoolean()获取下一个boolean数据
    • 如果输入数据不匹配，会报InputMisMatchException错误

  9. switch后边使用的表达式可以是哪些数据类型

    • byte short char int 枚举类型 String类型

  10. for循环中，判断是否停止的条件一般使用大于(左闭右开)

  11. 双层for循环输出，外层控制行数，内层控制列数

  12. 数组一旦初始化，长度就确定了，数组长度一旦确定，就不可修改。

    数组属于引用类型，数组的元素也可以是引用类型，引用类型变量的值只有两种情况，null或者地址值

  13. 不同类型一维数组元素的默认初始化值

    • 整型：0
    • 浮点型：0.0
    • char：0
    • boolean：false
    • 引用类型：null

  14. 数据的存储结构

    • 线性表：顺序表（比如：数组）、链表、栈、队列
    • 树形结构：二叉树
    • 图形结构

  15. 算法：排序算法、搜索算法
