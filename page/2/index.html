一、入门

  1. JDK和JRE

    • JDK：JRE + Java核心开发工具(javac.exe、java.exe、javadoc.exe)
    • JRE：JVM + Java核心类库

  2. 为什么配置Path环境变量呢？

    • 刚安装好的JAVA只能在其bin目录下执行java命令，为了让电脑上任何地方都可以执行Java命令，就得配置Path环境变量

    • JAVA_HOME = bin的上一层目录

      path = JAVA_HOME%\bin

  3. 编译和运行.java文件

    • 编译：javac abc.java 编译错误，一般是语法问题
    • 运行：java abc

  4. char c = ‘’; //空字符，编译不通过

    String s = “”; //空字符串，能够正常运行

  5. 标识符命名规范：

    • 包名：aabbcc 全部小写
    • 类名、接口名：AaBbCc 单词首字母大写
    • 变量名、方法名：aaBbCc 除首字母外其他首字母大写 即驼峰命名法
    • 常量名：AA_BB_CC 全大写，单词间用下划线_隔开

  6. 八种基本类型变量、引用数据类型：

    • boolean、byte(1)、char(2) 、short(2)、int(4)、long(8)、float(4)、double(8)
    • 类、接口、数组

  7. 数据类型之间的自动转换(小到大)

    • byte、shrot、char->int->long->float->double

  8. 获取系统输入：

    • Scanner sc = new Scanner(System.in)
      • sc.next()获取下一个字符串
      • sc.nextInt()获取下一个int数据
      • sc.nextDouble()获取下一个double数据
      • sc.nextBoolean()获取下一个boolean数据
    • 如果输入数据不匹配，会报InputMisMatchException错误

  9. switch后边使用的表达式可以是哪些数据类型

    • byte short char int 枚举类型 String类型

  10. for循环中，判断是否停止的条件一般使用大于(左闭右开)

  11. 双层for循环输出，外层控制行数，内层控制列数

  12. 数组一旦初始化，长度就确定了，数组长度一旦确定，就不可修改。

    数组属于引用类型，数组的元素也可以是引用类型，引用类型变量的值只有两种情况，null或者地址值

  13. 不同类型一维数组元素的默认初始化值

    • 整型：0
    • 浮点型：0.0
    • char：0
    • boolean：false
    • 引用类型：null

  14. 数据的存储结构

    • 线性表：顺序表（比如：数组）、链表、栈、队列
    • 树形结构：二叉树
    • 图形结构

  15. 算法：排序算法、搜索算法

AJAX学习笔记

一、定义

AJAX = Asynchronous JavaScript and XML

即：异步的 JavaScript 和 XML

二、优点

在不重新加载整个页面的情况下，可以与服务器交换少量数据并更新部分网页内容！

三、前置知识

  • XMLHttpRequest 对象 (异步的与服务器交换数据)
  • JavaScript/DOM (信息显示/交互)
  • CSS (给数据定义样式)
  • XML (作为转换数据的格式)

四、工作原理

XMLHttpRequest是AJAX基础。

XMLHttpRequest是什么？

它用于在后台与服务器交换数据！是动态更新局部网页的基础。

目前市面上大部分浏览器均支持XMLHttpRequest对象。

创建XMLHttpRequest对象
向服务器发送请求

使用XMLHttpRequest对象的open()和send()方法：

方法 描述
open(method,url,async) 规定请求的类型、URL 以及是否异步处理请求。method：请求的类型；GET 或 POSTurl：文件在服务器上的位置async：true（异步）或 false（同步）
send(string) 将请求发送到服务器。string：仅用于 POST 请求

使用POST的情况：

  • 不愿使用缓存文件（更新服务器上的文件或数据库）
  • 向服务器发送大量数据（POST 没有数据量限制）
  • 发送包含未知字符的用户输入时，POST 比 GET 更稳定也更可靠
获取服务器响应
属性 描述
responseText 获得字符串形式的响应数据。
responseXML 获得 XML 形式的响应数据。
onreadystatechange事件

当 readyState 改变时，就会触发 onreadystatechange 事件，readyState 属性存有 XMLHttpRequest 的状态信息。

属性 描述
onreadystatechange 存储函数（或函数名），每当 readyState 属性改变时，就会调用该函数。
readyState 存有 XMLHttpRequest 的状态。从 0 到 4 发生变化。
0: 请求未初始化
1: 服务器连接已建立
2: 请求已接收
3: 请求处理中
4: 请求已完成，且响应已就绪
status 200: “OK” 404: 未找到页面

当 readyState 等于 4 且状态为 200 时，表示响应已就绪：

注意： onreadystatechange 事件被触发 4 次（0 - 4）, 分别是： 0-1、1-2、2-3、3-4，对应着 readyState 的每个变化。

解决联想拯救者Y7000电池为0的问题

发表于 2021-10-25 分类于 电脑维修
  • 电脑状况：早上睡醒之后正常开机，发现电池电量为0%，并显示正在充电，且一直保持该状态
  • 尝试修复：查了百度，分别使用了①放静电，长按电源键30s再松开②更新电池固件，该方法后很多人成功了，但是我没有
  • 真正的解决方法：突发奇想拔掉了电源适配器，发现电脑居然没自动关机，且电池显示状态为0%，这就好办了。在不插电的情况下，把电池电量耗尽(放个电源或者直播都行)，让它自动关机，然后接上充电器，成功秒杀~~(没错我是老厮的粉丝)

算法第五天

发表于 2021-09-08 更新于 2021-09-09 分类于 算法合集

第五天：根据代码随想录，程序员carl的指南刷题

  • 数组：

    • 二分查找：力扣704

      解题思路：正常二分法即可。注：left<=right , right = mid -1; left = mid+1;

    • 移除元素：力扣27

      解题思路：设原数组长度L，遍历整个数组，遇到指定元素，不放回数组并且L–，最后返回L就是长度。

    • 有序数组的平方：力扣977

      解题思路：0为分界，因为有序，所以只需要比较最左边和最右边的平方哪个大，然后将大的放在数组结尾就好了。需三个指针，一个指向开始，另一个指向结尾，还有一个从结尾到开始。

    • 长度最小的子数组：力扣209

      解题思路：滑动窗口，末尾不越界时，进入第一个while，end指针向后移动，直到计算和满足target的值；当和不小于target时，开始第二个while，获取最小子数组的长度，继续计算和，start指针向后移动。

    • 螺旋矩阵Ⅱ：力扣59

      解题思路：模拟从左到右，从上到下，从有到左，从下到上整个循环。先设置矩阵余量和矩阵四个角的大小。l =0,r=n-1,top=0,bot=n-1。当矩阵内还有数，while循环按顺序模拟四个操作上右下左top++,right–,bot–,left++，每次移动，矩阵内余量-1。

  • 链表

    • 移除链表元素：力扣203

      解题思路：设置虚拟头结点再进行删除操作，之后从虚拟头结点开始向后遍历，直到节点为空，如果中途遇到节点值为val，将其删除，否则，继续向后移动。
